# 🧮 Simple Calculator

A modern desktop calculator built with **C#** and **Windows Forms**, featuring a clean user interface powered by **Guna UI2**. The application supports both basic and advanced mathematical operations while providing a smooth and user-friendly experience.

## 📖 About the Project

This project is a desktop calculator application developed using Windows Forms. It performs common arithmetic operations along with several additional mathematical functions. The application was designed with a modern interface using **Guna UI2** and focuses on clean code organization by separating the calculator logic into reusable methods.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 📊 Percentage calculation
- % Modulus operation
- √ Square root
- x² Square calculation
- 🔢 Decimal number support
- ± Switch between positive and negative numbers
- ⌫ Backspace (delete the last entered digit)
- 🧹 Clear current input
- ♻️ Reset the calculator state
- 🌙 Toggle between Light Mode and Dark Mode
- 🎨 Modern interface built with Guna UI2
- ⚡ Fast and responsive user experience

## 🛠️ Technologies Used

- C#
- .NET Framework
- Windows Forms
- Guna UI2
- Object-Oriented Programming (OOP)
